# PoTM Portable Tool (P1)

**What this is:** A single-file, tag-driven tool stance (no profiling, no logs, no audits).  
**How to use:** Paste `potm_portal_tool.md` into any model’s system/instructions.  
**Trigger words:** `menu`, `two_pass`, lenses (`EDGE`, `INTUIT`, `CONTRARY`, `OPENQ`, `CAST`, etc.).  
**Out of scope:** MSRL, multi-agent governance, adaptive profiling — see Agent edition.
